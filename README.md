# Amazing Deals India 🔥🇮🇳

Daily Amazon India deals website with Instagram integration

## Features

✅ **Daily Amazon India deals under ₹5000**  
✅ **Instagram-style grid display**  
✅ **Admin dashboard for post management**  
✅ **Automated daily scraping**  
✅ **Amazon Associates India integration**  

## Quick Setup

1. Update `data/config.json` with your Amazon India affiliate tag
2. Add `APIFY_TOKEN` to GitHub Secrets
3. Deploy to Vercel
4. Automation runs daily at 9:30 AM IST

See `AUTOMATION_SETUP.md` for detailed instructions.
